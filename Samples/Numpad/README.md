|:--------------|:---------------------------------------------|

| Siv3D Version | v0.6.13                                      |

## 説明 | Description

タッチやクリック、キーボードで数値を入力できるパッドを表現する `Numpad` クラスです。  
色やサイズ、フォントや挙動をカスタマイズできます。
